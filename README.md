# 使用 
访问xxx
* 输入你的openai key，这个key只存储到你的本地，服务器也不会存储，完全安全
* 生成: 右下角输入你的需求然后点击生成等待生成完成
* 二次修改: 鼠标选中要修改的地方，然后点击，之后在右下角的输入框中输入你想要修改的内容，点击修改，之后点击生成等待生成完成
